You want to learn how to make quality web design from scratch, but do not know where to start? Welcome to WEB DESIGN. BASIC.


For two months of intensive work you will get acquainted with the key principles in the work of a web designer. In addition to the basics of web design, you will learn what a modular grid is, how to apply it, how to create an adaptive design, and all this will be in practice.


The acquired skills and experience, will be an excellent base for starting a career as a web designer
